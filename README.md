# UCSG

2D game about building a fleet of spacecraft to control a solar system

Player awakes in an escape pod in a foreign solar system, orbiting a planets moon.
Player must perform spacewalks to harvest sattelite debris and asteroids to gather materials.
Use materials to expand the escape pod.

## Focusing in Mechanics
For the alpha build, the following features are good enough I think:

### Must Have
- Small, expandable spacecraft.
- Can Spacewalk in the vacuum to retrieve debris + asteroids.
- Crafting system to produce buildings/items by hand

### Should Have
- Technology research in the form of delivering resources to the ship computer
- Survival pressure (food, water, oxygen)

### Could Have
- Light Mod Support
